Notes on the Visualization

This graph shows a PCA projection of the original embedding space.
Dimensionality Reduction
PCA reduces 1536-dimensional embeddings to 2D or 3D so humans can inspect their relative positions.
Interpretation & Retrieval
The visualization is intended for interpretation only. Similarity search in this project is performed using cosine similarity on the original embeddings, not on the projected coordinates shown above.
Spatial Distortion
As with a flat map of the Earth, projecting a high-dimensional space inevitably introduces distortion. Nearby points usually indicate semantic similarity, but the displayed distances should not be treated as exact.
